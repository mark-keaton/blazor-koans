# Task: Beginner Koans

## Description
Create 16 beginner-level koans covering Blazor fundamentals: Components, Data Binding, Events, and Lifecycle. Each topic gets ~4 koans with progressive complexity. Include skeleton components and solution comments.

## Acceptance Criteria
- [ ] 4 koans for Components (creating, parameters, child content, render fragments)
- [ ] 4 koans for Data Binding (one-way, two-way, bind modifiers, bind:after)
- [ ] 4 koans for Events (onclick, EventCallback, EventCallback<T>, prevent default)
- [ ] 4 koans for Lifecycle (OnInitialized, OnParametersSet, OnAfterRender, IDisposable)
- [ ] All tests have `[Trait("Category", "Beginner")]`
- [ ] All tests fail with placeholder values
- [ ] All tests pass when solutions are applied
- [ ] Corresponding skeleton components exist in `src/`
